Welcome to the Tao of testing online calculator FAQ

You can use our calculator for a variety of simple maths related tasks.

What features does the calculator have

  1. Supports addition, subtraction, multiplication and division operations of integer, real and floating point numbers.
  2. Displays a complete history of opertaions for the calculations made

  3. Memory features:

    • Memory Add - adds the current value to the value stored in memory
    • Memory Minus - adds the value to the value stored in memory
    • Memory Clear - nukes the current memory back to zero
    • Memory Recall - replaces the current value with one in the memory.
  4. Calculation overflow errors are detected and displayed to users
  5. Comprehensive help
  6. Version history
  7. Linked to bug tracking systems on github.
  8. Calculation overflow errors are detected and displayed to users
  9. Supports Pi to 13 d.p.

How to use the calculator history

A session with our calculator lasts until you exit the page. The calculator history saves all the calculations performed. You are able to edit the history to calculate new calculations.
